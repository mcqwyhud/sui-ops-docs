# SUI-Ops 商业系统快速开始

欢迎使用 SUI-Ops 分布式经营系统。本指南旨在引导您完成从基础设施筹备到业务上线全流程。

## 业务架构简述
本系统采用分布式微内核架构，由 **[控制总控]** 与 **[节点网关]** 协同工作。

| 模块 | 核心职责 |
| :--- | :--- |
| **控制总控** | 负责资产编排、财务结算、分销裂变逻辑等核心运营决策。 |
| **节点网关** | 负责高并发流量转发、协议执行及实时链路质量探测。 |

---

## 🚀 第一步：合作与基础设施筹备

本系统采用 SaaS 商业模式，在正式启动业务前，请悉知以下流程：

1. **联系购买**：联系官方技术支持获取服务授权与套餐确认。
2. **总控 VPS 筹备**：
   - 您需自主承担总控服务器费用。
   - 我们将协助您在选定的 VPS 提供商处完成系统部署。
   - **合作声明**：总控服务器的租用、维护及续费由您自行负责，我方提供系统运维支持，不涉及第三方费用代垫。
3. **环境交付**：部署完成后，我们将交付专属管理后台入口。

---

## 🛡️ 第二步：高防安全架构

本系统深度集成安全防御体系，保障您的业务长效稳定：

> **核心安全策略**：系统全站支持接入 Cloudflare 等高防 CDN。通过源站隐藏技术，我们将您的 **[前端门户]** 与 **[总控后台]** 置于绝对匿名状态，从根本上免疫 DDoS/CC 攻击。

- **分布式解耦**：支持将业务服与管理总控分离，外界无法通过任何手段溯源您的真实总控 IP。

---

## 🛠️ 第三步：客户端与接入前置要求

1. **域名接入（核心强制要求）**：
   - 客户端对接**必须使用域名**。严禁直接使用 IP 地址对接。
   - **风险提示**：因未使用域名导致 IP 变更引起客户端失效，属于二次打包定制范畴，需额外加收服务费。
2. **客户端定制**：提供品牌化、UI 个性化及协议级深度优化。
